# 課程簡介
本課程將介紹生物影像分析中的非監督式機器學習方法，包含降維與分群，並帶領學員使用 **napari** 與 **napari-clusters-plotter** 進行實作。內容包含：

- 降維方法（PCA、UMAP、t-SNE）
- 分群方法（K-means、HDBSCAN 等）
- 實際資料分析流程（segmentation、measurement、分析與視覺化）
