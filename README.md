📅 課程時程與地點
課程日期： 2026/3/9 至 2026/5/4（每週一，共 8 堂） 

課程時間： 15:00 - 17:00

上課地點： 台大醫學院圖書館二樓會議室（僅限實體錄取者，若為校外人士請提前至一樓櫃台辦理換證進入圖書館）
